# TOC

Generates a TOC refers to headings in posts.

![Preview of TOC](./img/toc.png)

```yaml filename="theme/hexo-theme-cupertino/_config.yml"
toc: true
```

Control the depth when generating, for example, to heading 2:

```yaml filename="theme/hexo-theme-cupertino/_config.yml"
toc_max_depth: 2
```
