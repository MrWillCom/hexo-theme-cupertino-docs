Rainbow banner is disabled by default. Enabling it will make it appear every June:

```yml filename="_config.cupertino.yml"
rainbow_banner: true
rainbow_banner_colors:
  - '#e50000'
  - '#ff8d00'
  - '#ffee00'
  - '#008121'
  - '#004cff'
  - '#760188'
rainbow_banner_shown: auto
rainbow_banner_month: 6
```

`rainbow_banner_colors` accepts an array of CSS colors. `rainbow_banner_shown` accepts `auto` or `always`.

To to display rainbow banner all year long, change `rainbow_banner_shown` to `always`.
