# Nav Blur Gradient

Nav blur gradient is an elegant effect for the navigation, immersing the visitors in the main content. This gradient is stacked with 6 blur gradient layers to balance performance and fineness.

![Screenshot of nav blur gradient](./img/nav-blur-gradient.jpg)

Nav blur gradient can be turned on by setting:

```yml filename="_config.cupertino.yml"
nav_blur_gradient: true
```
