# Metadata

## Favicon

Your blog's icon, favicon is the icon shown on the browser tab.

![Favicon shown on the browser tab](./img/favicon.jpg)

```yml filename="_config.cupertino.yml"
favicon: <path-to-favicon>
```

This value is directly given to [`favicon_tag`](https://hexo.io/docs/helpers#favicon-tag).

## Manifest

Path to [`manifest.json`](https://developer.mozilla.org/docs/Web/Manifest), mostly used by PWA.

```yml filename="_config.cupertino.yml"
manifest: <path-to-manifest>
```
